20201203 建立Hugo流程

Hugo Blog建置過程

  1. 安裝hugo => 下載完hugo 設定環境變數(Window) / brew install hugo(macOS)
  2. 檢查版本 hugo version
  3. hugo new site (你的blog名稱)
  4. cd (你的blog名稱)/
  5. git clone theme =>例如此blog目前是用m10c
  6. cd/themes =>檢查一下
  7. 回到根目錄=> cd (你的blog名稱)/
  8. 設定config.toml 加上 theme = “m10c”
  9. 在根目錄下執行 hugo server -t m10c –buildDrafts =>啟動server
  10. 創建一個文章 hugo new post/blog.md
  11. hugo –theme=m10c –baseUrl=“你的git repository” –buildDrafts
  12. cd public/ 執行git init
  13. git add .
  14. git commit -m “標題” => 在 Git 提交(commit)檔案
  15. git remote add origin “git的url”
  16. git push
  17. git push -u origin master (輸入帳密)即完成
